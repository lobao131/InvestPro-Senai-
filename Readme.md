# InvestPro - Simulador de Renda Fixa

## 1. Descrição do Sistema (Mini-Mundo)
O **InvestPro** é uma aplicação web focada em educação financeira e projeção de investimentos. Diferente de calculadoras comuns, ele resolve a dor do investidor iniciante de não compreender a "sopa de letrinhas" do mercado (CDI, SELIC, IPCA). 

O sistema permite que o usuário insira um valor e um prazo, calculando automaticamente o retorno líquido em quatro modalidades (Poupança, Tesouro Selic, CDB e IPCA+), descontando o Imposto de Renda Regressivo. O grande diferencial técnico é o cálculo do **Ganho Real**, demonstrando visualmente se o investimento superou ou perdeu para a inflação.

## 2. Tecnologias e Funcionalidades
**Tecnologias:**
* HTML5 (Estrutura Semântica)
* CSS3 (Grid Layout e Responsividade)
* JavaScript (Manipulação do DOM e Lógica Financeira)

**Funcionalidades:**
* Simulação comparativa em tempo real.
* Cálculo automático de Imposto de Renda (Tabela Regressiva).
* Indicador visual de Ganho Real vs. Inflação.
* Cards educativos explicativos sobre benchmarks financeiros.

## 3. Procedimentos de Execução Local
1.  Faça o download do arquivo `.zip` ou clone o repositório.
2.  Extraia os arquivos em uma pasta local.
3.  Abra o arquivo `index.html` em qualquer navegador moderno (Chrome, Firefox, Edge).
4.  Não é necessário instalação de servidores ou dependências externas.
